Локальный тип временри -> день, месяц, год
Локальный тип временри -> время без даты и часового пояса
Локальный тип временри -> клмюинация даты и времени

Часовые пояса
1. ZoneOffset --> смещение по времени от +14 до -12
2. ZoneId --> Id часового пояса Европа/Берлин
3. ZoneRules --> правила смены часовыъ поясо( летнее и зимнее времена)
